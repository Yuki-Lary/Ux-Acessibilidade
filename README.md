# UX e Acessibilidade

Projeto educativo sobre desenvolvimento de interfaces inclusivas. Esta página mostra como aplicar boas práticas de design de experiência do usuário e acessibilidade, incluindo:

- Navegação por teclado
- Contraste de cores legível
- Formulários acessíveis com labels e mensagens claras
- Estrutura semântica em HTML
- Feedback dinâmico com `aria-live`

## Como usar

1. Abra `index.html` em seu navegador.
2. Use a tecla <kbd>Tab</kbd> para navegar pelos links e campos do formulário.
3. Ative o modo de alto contraste clicando no botão no cabeçalho.
4. Preencha o formulário para ver o feedback acessível.

## Arquivos

- `index.html`: estrutura e conteúdo do site.
- `style.css`: estilos responsivos e acessíveis.
- `script.js`: comportamento de alto contraste e validação de formulário.
